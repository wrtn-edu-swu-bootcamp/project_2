## 🎯 서비스 핵심 요약

### 서비스명
**밥심(BAB:SSIM)** - Balance And Budget Smart Meal

### 핵심 가치
> "3분 질문으로 오늘의 영양 만점 한 끼를 찾아드립니다"

### 타겟 사용자
자취하는 대학생 (19-26세)

### 핵심 기능
1. 간단한 질문 응답 (3-7개 질문, 3분 이내)
2. 개인 맞춤 식단 3개 추천
3. 상세 레시피 및 영양 정보 제공

### 필수 질문 항목
- ✅ 조리 가능 여부 (시간, 도구, 경험)
- ✅ 예산 (한 끼당 예산 범위)
- ✅ 기피 식재료 (알레르기, 개인 취향)

### 차별화 포인트
- 검색 불필요한 능동적 추천
- 자취생에 특화 (1인분, 저예산, 간단 조리)
- 영양 균형 우선 고려
- 빠른 의사결정 지원

## 📊 MVP 성공 기준 (3개월)

- 활성 사용자(MAU): 500명 이상
- 재사용률: 50% 이상
- 만족도: 4.0/5.0 이상
- 조리 시도율: 60% 이상

### 궁극적 검증 질문
1. 사용자가 추천받은 메뉴를 실제로 만들어 먹는가?
2. 사용자의 영양 섭취가 개선되는가?
3. 사용자가 지속적으로 사용하는가?
4. 사용자가 주변에 추천하는가?

→ **4가지 모두 YES = MVP 성공**
